# Roshan System

## Project Overview
Roshan System is a comprehensive application built using C# WPF for the NewRoshanSystem and Python Tkinter for the OldRoshanSystem.

### NewRoshanSystem (C# WPF)
The NewRoshanSystem includes the following main components:
- **MainWindow**: The primary interface of the application.
- **Browser**: A web browsing component integrated into the application.
- **Notepad**: A simple text editing tool.
- **Calculator**: A basic calculator for performing arithmetic operations.
- **Overclocker**: A component designed for overclocking functionalities.
- **Run**: A runner for executing various tasks and commands.
- **StartMenu**: The main menu for navigation and launching different components.
- **winver**: Displays the Windows version information.

### OldRoshanSystem (Python Tkinter)
The OldRoshanSystem is structured as follows:
- **main.py**: The entry point of the Tkinter application.
- **functions**: A module containing various helper functions.
- **gui**: A module containing the graphical user interface components that render the application.
