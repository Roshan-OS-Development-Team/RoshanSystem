## Project Structure

### New Roshan System (C# WPF)
- **MainWindow.xaml**: The main window of the WPF application.
- **App.xaml**: Application entry point for the WPF app.
- **Models/**: Contains data models used in the application.
- **Views/**: Contains XAML files and corresponding view models.
- **ViewModels/**: Contains the view models that handle logic and data for the views.

### Old Roshan System (Python Tkinter)
- **main.py**: The main execution file for the Tkinter application.
- **gui/**: Contains the graphical user interface files.
- **controllers/**: Contains the logic to control the GUI.
- **models/**: Contains data models used in the Tkinter application.
